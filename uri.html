URI

Stands for Universal Resource Identifier; it is essentially a uniform way to access a set of information (resource), usually through a string of letters and numbers. URLs are similar to URIs, but a URI can be any part of a URL (a relative URI) or the whole thing (an aboslute URI). Can be used to describe either a file's name or location, or both. Definition adapted from w3.org 2019 and TechTerms by Jay Townsend
